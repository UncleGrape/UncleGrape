# UncleGrape
UncleGrape is desiged to be able to be run from shared web hosts.

## Modern Features
* Manual, Automatic, or DeadMan Kill Switch *
* Custom Port Relay *
* Scheduled Access *
* Ephemeral Account Provisioning *
* Cubesat Space Protocol (CSP) *
* Fast Deployment *
* TCP/UDP/SCTP *
* SSL *
* IPV6 *
* Internet over Ping (IoP)
* TOR *
* Support for dApps *
* Intrusion Detection *
* Extension API *

*Planned
